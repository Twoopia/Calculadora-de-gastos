## 2. Justificativa

O projeto foi proposto como trabalho final da disciplina de Gerência de Projetos, com o objetivo de aplicar na prática os conceitos de metodologia Scrum aprendidos ao longo do semestre. O tema "controle financeiro pessoal" foi escolhido pela equipe por ser:

- Relevante para o público universitário (o próprio time)
- De escopo adequado para 4 sprints
- Capaz de demonstrar todas as funcionalidades de um sistema web real

---

## 3. Objetivos do Projeto

**Objetivo principal:**  
Desenvolver e entregar o sistema SmartExpense até 02/07/2026, com todas as funcionalidades especificadas no Product Backlog.

**Objetivos secundários:**
1. Aplicar o framework Scrum conforme o Guia Scrum 2020
2. Demonstrar conhecimento em desenvolvimento full-stack (Python/FastAPI + HTML/CSS/JS)
3. Manter repositório Git organizado com boas práticas de versionamento
4. Produzir documentação técnica e de processo completa
